## Java

Intellij idea, 只要是一名合格的Java程序员就应该知道的一款集大成的IDE。良好的代码提示, 友好的交互界面将大部分的Eclipse用户吸引过来, 当然, 如果你想使用她来进行web开发, 那么你如果有足够的能力, 还是希望支持正版, 但是学生肯定是没办法支付这么昂贵的价格的, 所以, 破解请看这里。
