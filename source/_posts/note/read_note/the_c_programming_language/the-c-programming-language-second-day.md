# 函数

## 函数的声明

需要在文件开头声明函数的返回类型, 函数名以及形参, 不需要方法体, 这一点c++和c差不多。
比如:

```c
int power(int m, int n);

/**
 * 计算幂
 * @param m 基数
 * @param n 幂方
 */
int power(int m, int n)
{
  int p = 1;
  for (int i = 0; i != n; ++i)
  {
    p *= m;
  }
  return p;
}
```

## 形式参数

所有的函数参数都是通过值传递的, 如果需要在函数中改变主函数的变量, 可以向函数传递指针。
**数组函数传递的是数组起始元素的位置或地址。**

## 字符数组

> hello\0

这样的字符串常量时, 它将以字符数组存储
