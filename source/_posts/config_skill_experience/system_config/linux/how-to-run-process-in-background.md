# 使用 nohup & 来保持后台运行

在执行命令的时候, 在命令末尾加一个 `&` 符号来让这个命令保持后台运行。

比如:

```shell
nohup make -j2 &
```

在这样做了之后, 按 `ctrl-c` 回到命令行界面, 使用 exit 命令来关闭当前session, 这样你的后台命令还是会保持运行。

# 使用 ctrl-z 和 bg 后台运行

当你的程序已经启动的时候, 可以先使用 `ctrl-z` 将当前程序放到后台并暂停运行, 之后使用 `jobs` 来查看序号, 之后使用 `bg 序号` 来使程序后台运行。
